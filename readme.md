#### 国际单位制 (SI)
three.js 处处都使用的国际单位制
* SI 单位
    * 距离以米为单位（three.js 的 1 单位 = 1 米）。
    * 时间以秒为单位
    * 光照以国际光照强度单位测量， 坎德拉(cd)、流明 (lm) 和勒克斯 (lx)（至少你需开启renderer.physicallyCorrectLights）。
